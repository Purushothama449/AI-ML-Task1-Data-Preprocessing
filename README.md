

## 📌 Task: Data Cleaning & Preprocessing

## 🎯 Objective
To clean and prepare raw data for machine learning.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Tkinter (GUI)

## ⚙️ Steps Performed
- Loaded dataset using pandas
- Handled missing values (mean & mode)
- Converted categorical data into numerical
- Standardized numerical features
- Visualized outliers using boxplot
- Removed outliers using IQR method
- Built GUI to automate preprocessing

## 📊 Output
- Boxplot graphs (before & after outlier removal)
- Cleaned dataset generated
